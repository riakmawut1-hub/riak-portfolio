# Public Assets

Place your static assets here:

- **resume.pdf** - Your CV/Resume (required for download button)
- **favicon.ico** - Site favicon (optional)
- Any other images or static files

## Adding Favicon

1. Create or download a favicon.ico file
2. Place it in the `public/` directory
3. Next.js will automatically serve it at `/favicon.ico`

## Adding Resume

1. Export your CV as PDF
2. Name it `resume.pdf`
3. Place it in the `public/` directory
4. The download button in the contact section will automatically work
